Hyperslider

This is just a simple plugin slider.
(I need to add more features but I'm too busy with other stuff! :D)

The plugin is located here https://github.com/pogadz/hyperslider

Usage

By default it is set to "fade" effect with "6 seconds" duration.

Fade Effect



    $('#sample1').hyperslider({
        animation: 'fade',
        autoplay: true,
        delay: 5000
    });

Flash Effect



    $("#sample2").hyperslider({
       animation: "flash",
       autoplay: true,
       delay: 4000
    });

Slide Effect



    $('#sample3').hyperslider({
        animation: 'slide',
        autoplay: true,
        delay: 4000
    });

Other settings

You can also use the callback "before" and "after" settings



    $('#sample3').hyperslider({
        animation: 'slide',
        autoplay: true,
        delay: 4000,
        before: function(el, settings){
            // code before
            // the slider loads
        },
        after: function(el, settings){
            // code after
            // the slider loads
        }
    });

Note: You can add other data attributes settings like the example below.

  • data-hyperslider - this will automatically enable the plugin with it's default settings
  • data-animation="slide" - fade, flash, slide
  • data-autoplay="true" - true or false
  • data-delay="7000" - 1000 milliseconds
  • data-before="function() { console.log('I am fired first!') }" - callback before the slide
  • data-after="function() { console.log('I am fired after the slider loads!') }" callback after the slide
